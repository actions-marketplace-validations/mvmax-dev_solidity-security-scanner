# Security Policy

## Supported Versions

Currently, only the latest `main` branch of the Solidity Security Scanner is supported with security updates. 

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

Security is a core priority for this repository, as it is a security scanner itself. If you discover a vulnerability within this codebase, please **do not** open a public issue.

Instead, please report it via one of the following methods:
1. Email the core maintainers directly at security@mvmax-dev.org
2. Open a draft security advisory via GitHub Security Advisories for this repository.

### What to include in your report:
- A descriptive title
- Steps to reproduce the issue
- The potential impact of the vulnerability
- Any potential mitigation you suggest

We will acknowledge your report within 48 hours and provide a timeline for a fix. All reporters who follow this responsible disclosure policy will be credited in our release notes and Security Advisories.
